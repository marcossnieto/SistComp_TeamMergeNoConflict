# **Práctico 1 - Benchmarks**

---

## **Primera Parte**

Una prueba de rendimiento (en inglés *benchmark*) es una técnica utilizada para medir el rendimiento de un sistema o uno de sus componentes. Más formalmente puede entenderse que una prueba de rendimiento es el resultado de la ejecución de un programa informático o un conjunto de programas en una máquina, con el objetivo de estimar el rendimiento de un elemento concreto, y poder comparar los resultados con máquinas similares.

En el ámbito de las computadoras, una prueba de rendimiento podría ser realizada en cualquiera de sus componentes, ya sea la CPU, RAM, tarjeta gráfica, etc. También puede estar dirigida específicamente a una función dentro de un componente, como la unidad de coma flotante de la CPU, o incluso a otros programas. Los benchmarks devuelven información detallada de todas las características que posee y con base en dicha información, se puede evaluar si está completamente optimizado para correr o ejecutar las aplicaciones que necesitamos dependiendo del área.

### Benchmarks seleccionados

A continuación se presenta una lista de benchmarks considerados más útiles según las tareas diarias que realizamos, junto con el benchmark más representativo para cada una:

| Tareas | Benchmark más representativo |
|--------|------------------------------|
| Todo lo relacionado al gaming | 3DMark |
| Navegación web y multi-tareas | PCMark 10 |
| Optimización de software | Build Linux Kernel |
| Rendimiento general de CPU | Cinebench R23 |
| Lectura/escritura en disco | CrystalDiskMark |
| Rendimiento de memoria RAM | AIDA64 Memory Benchmark |


**3DMark** es el benchmark más elegido para medir el rendimiento gráfico en juegos. Simula cargas reales como renderizado 3D, iluminación y física, evaluando tanto la GPU como la CPU en escenarios típicos de gaming.

**PCMark 10** mide el rendimiento en tareas diarias como navegación web (carga de páginas, JavaScript), productividad (oficina) y multitarea, simulando un uso real del equipo.

**Build Linux Kernel** mide el tiempo de compilación del kernel de Linux, una tarea intensiva en CPU ideal para evaluar eficiencia en desarrollo, compilación y optimización de software, especialmente en uso de múltiples núcleos.

**Cinebench R23** evalúa el rendimiento de renderizado 3D de la CPU, tanto en un solo núcleo como en todos los núcleos simultáneamente. Es muy utilizado en comparativas de procesadores de escritorio y portátiles.

**CrystalDiskMark** mide la velocidad de lectura y escritura secuencial y aleatoria de discos SSD y HDD, siendo especialmente útil para evaluar el impacto del almacenamiento en tareas como edición de video o bases de datos.

**AIDA64 Memory Benchmark** evalúa el ancho de banda y la latencia de la memoria RAM, métricas clave para tareas que dependen de grandes volúmenes de datos en memoria, como procesamiento de imágenes o simulaciones.
